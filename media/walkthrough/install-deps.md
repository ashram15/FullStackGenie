# Install Dependencies

Before you can run your project, you need to install the dependencies.

## Frontend:

```bash
cd frontend
npm install
```

This will install:
- React
- Vite
- Axios
- All development dependencies

## Backend:

```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

This will install:
- FastAPI
- Uvicorn
- Python-dotenv
- And other dependencies

⏱️ **Estimated time:** 2-3 minutes
